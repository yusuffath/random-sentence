# **App Name**: Sentence Explorer

## Core Features:

- Sentence Listing: Fetches and displays a list of sentences from a public API.
- Search Redirection: Redirects the user to a Bing search using the selected sentence as the query.
- Tracking click: Remembers sentences previously clicked by the user.
- Sentence Regeneration: Generates new list of sentences using an LLM tool after user click this action

## Style Guidelines:

- Primary color: HSL(210, 60%, 50%) – a vibrant blue, converted to hex: #3399FF
- Background color: HSL(210, 20%, 95%) – a light blueish gray, converted to hex: #F0F4FF
- Accent color: HSL(180, 60%, 40%) – a turquoise-green, converted to hex: #33CCCC
- Body and headline font: 'PT Sans', a humanist sans-serif. Note: currently only Google Fonts are supported.
- Minimalist line icons for actions and navigation.
- Clean, card-based layout for displaying sentences.
- Subtle fade-in animations when loading new sentences.
